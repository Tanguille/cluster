# Media Context

**When to use:** Recyclarr, Trash Guides, custom formats, or media automation.

- `select_all: true` imports all custom formats from Trash Guides regardless of default status.
- `delete_old_custom_formats` is deprecated in Recyclarr v8.4.0; automatic replacement is now the default.
- Custom-format YAML files can contain duplicate `trash_ids`, with and without scores; they can be consolidated.
- Dual Audio means Japanese and English, not Japanese only.
